Projects

Group related stacks together under a single project.

What is a project?

A project lives inside an organization and groups one or more stacks together. For example, you might have a project called "My SaaS App" containing a MERN web stack and an LLM stack.

Creating a project

From your organization dashboard, click Create Project. Give it a descriptive name — you can always rename it later from the project settings page.

Adding stacks

Inside your project, open the Stack Marketplace to browse and deploy stacks. Each stack is deployed as independent cloud infrastructure but managed together within your project.

Project transfers

Need to move a project to a different organization? Deploy Box supports project transfers — the receiving organization's owner will get an invitation to accept the transfer.

Project settings

From the project settings page you can:

  • Rename your project
  • Transfer the project to another organization
  • View and manage all stacks within the project
