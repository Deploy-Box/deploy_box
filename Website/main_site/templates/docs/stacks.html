What is a stack?

A stack is a complete, pre-configured set of cloud resources and services that form a working application environment. When you deploy a stack, Deploy Box uses Infrastructure-as-Code to provision everything automatically — databases, compute, networking, CI/CD, and more.

Stack lifecycle

Every stack goes through these stages:

  1. Provisioning — Cloud resources are created via Terraform.
  2. Active — Your stack is live and ready to use.
  3. Updating — Configuration changes are being applied.
  4. Destroying — Resources are being torn down and cleaned up.

Managing your stack

From your stack dashboard you can:

  • View real-time deployment status and logs
  • Configure environment variables for each environment
  • Manage settings like stack name and configuration
  • Access your stack's GitHub repositories
