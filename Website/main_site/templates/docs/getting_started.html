Quick Start Guide

Get from zero to a running stack in just a few minutes.

1

Create your account

Head to the sign-up page and create your account. Deploy Box uses WorkOS for authentication, so you can sign up with your email or an SSO provider.

2

Create an organization

After signing in you'll land on the Dashboard. Create your first organization — this is the top-level container for all your projects and team members.

Tip: You can create multiple organizations to separate personal projects from client work.

3

Create a project

Inside your organization, create a project. Projects group related stacks together — for example, a "My SaaS App" project might contain a web stack and an AI stack.

4

Deploy a stack

Open the Stack Marketplace from your project dashboard and pick a stack — MERN, Django, Mobile, AI/ML, or any other available option. Click Deploy and Deploy Box will provision all the cloud infrastructure for you automatically.

Deployment typically takes a few minutes. You can monitor progress in real time from the stack dashboard.

5

Connect your code

Once your stack is live, link your GitHub account from your profile settings. Deploy Box will create repositories for your stack and set up CI/CD pipelines automatically.
