Docs API Reference

API Reference

Integrate with Deploy Box programmatically using our REST API.

Base URL

https://api.deploybox.io/v1

All API endpoints are served under the /v1 namespace.

Authentication

The Deploy Box API supports two authentication methods:

API Key

Include your API key in the request header:

Authorization: Api-Key your-api-key-here

Microsoft Entra ID (OAuth 2.0)

For enterprise integrations, authenticate using a bearer token from Microsoft Entra ID.

Available endpoints

Method Endpoint Description
GET /v1/weather Get current weather data
GET /v1/geocoding Geolocation lookup
POST /v1/chatgpt ChatGPT integration
POST /v1/email Send transactional email
GET /v1/self-service Client self-service portal

Rate limiting

API requests are rate-limited to ensure fair usage. When you exceed the limit, the API returns a 429 Too Many Requests response with a Retry-After header.

Error handling

The API returns standard HTTP status codes. Error responses include a JSON body with details:

{
  "error": "Unauthorized",
  "message": "Invalid or missing API key",
  "status": 401
}
